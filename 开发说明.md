每个文件夹中应包含如下文件：
* <文件夹\>file：额外文件，例如markdown使用图像等
* file@-filename-：文献
* bib@-filename-.bib：bibtex文件
* comments@-filename-.md：批注
* info@-filename-.json：信息

info file 包含：
* device_import：导入电脑？
* time_import: 文献导入时间
* time_modify: 修改时间(上次打开文件/bib/comment/misc文件夹的时间)
* tags：标签
* ...
